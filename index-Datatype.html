Tipos de Dados: As Gavetas da Biblioteca

Pense nos tipos de dados como diferentes tipos de gavetas para guardar informações. Cada gaveta tem um formato específico para o que vai dentro: uma para números, outra para textos, outra para datas, etc. Escolher a gaveta certa é fundamental para manter tudo organizado.

Números

  • INT: Para números inteiros. Ex: idade = 25
  • BIGINT: Para números inteiros muito grandes. Ex: CPF = 12345678901
  • DECIMAL(10,2): Para números com vírgula e precisão exata, como dinheiro. Ex: preço = 49.90
  • FLOAT/DOUBLE: Para números com vírgula e precisão aproximada. Ex: peso = 1.75
  • BOOLEAN: Para valores de verdadeiro ou falso (geralmente 1 ou 0). Ex: ativo = 1

Data e Hora

  • DATE: Para armazenar apenas a data. Ex: 2025-08-19
  • DATETIME: Para armazenar data e hora juntas. Ex: 2025-08-19 15:30:00
  • TIMESTAMP: Similar ao DATETIME, mas é atualizado automaticamente em algumas operações.
  • TIME: Para armazenar apenas a hora. Ex: 15:30:00
  • YEAR: Para armazenar apenas o ano. Ex: 2025

Texto

  • CHAR(n): Para textos de tamanho fixo. Ex: código = 'AB123'
  • VARCHAR(n): Para textos de tamanho variável (mais comum). Ex: nome = 'Thierry'
  • TEXT: Para textos muito grandes, como descrições. Ex: descrição = "Produto..."
  • ENUM: Para escolher uma única opção de uma lista. Ex: status = 'sim'
  • SET: Para escolher várias opções de uma lista. Ex: hobbies = 'a,c'

Arquivos

  • BLOB: Para armazenar dados binários grandes, como fotos, PDFs ou músicas. Ex: foto_de_perfil
